HTML Dom Renderers PkTree & PkTreePainter

Introduction

These classes render HTML, building DOM trees, & rendering them as HTML.

PkTreePainter extends PkTree, but different intentions.

PkTree can be used to build DOM Trees, built up & nested.

PkTreePainter: Used to build small, configurable components - buttons, forms, sections.

PkTree

Extends PartialSet - builds any DOM element & inserts it into the tree. 

$pkt = new PkTree();
$level = $pkt->div($content1,$attributes1)->h2($content2,$attributes2);
#h2 is inside div
#$level is at the node h2
#$pkt is at base
#adding to $level puts content inside h2
#addint to $pkt puts content at base level
#But it all goes into $pkt when you return it

$pkt::div() creates a NEW PkTree

#Be careful NOT to return $level - or else you just get the subset!
